# Step2 试卷结构改版 — 统一结构 + 五种模板

**日期**: 2026-06-04
**状态**: 设计完成，待评审

---

## 1. 背景与动机

当前 Step2（试卷结构）存在两种"模式"：按题型分组和按单元分区，通过 Radio.Group 切换。问题：

- 两种模式底层共享同一数据模型（unit + question_config），UI 却给用户"两套系统"的错觉
- "按题型分组"本质是"按单元分区"的特例（每个题型=一个单元），不应作为平行模式
- 单元语义不明确，教师不知何时用哪种模式

## 2. 核心决策

| # | 决策 | 说明 |
|---|------|------|
| 1 | **单元 = 通用题目分组容器** | 系统不限制语义，教师自由定义 |
| 2 | **统一结构，去掉模式切换** | Step2 始终是单元列表 |
| 3 | **五种语义模板** | 新建时选择，决定初始结构 + 纸质排版 + 在线答题行为 |
| 4 | **模板在新建时选定，编辑时不可变更** | 简化设计，避免数据迁移复杂度 |
| 5 | **Step2 单页** | 模板选择 + 单元编辑 + 汇总栏同页 |

## 3. 五种语义模板

### 汇总对比

| 维度 | A 知识模块 | B 题型 | C 难度递进 | D 卷别 | E 通用 |
|------|-----------|--------|-----------|--------|-------|
| 单元数 | 可增减 | **固定1** | 可增减(默认3) | **固定2** | 可增减 |
| 单元名在试卷中 | 显示 | **不显示** | 显示+难度标签 | 显示 | 显示 |
| Step2编辑视图 | 卡片式 | **平表式** | 卡片式 | 卡片式 | 卡片式 |
| 题号 | 跨单元连续 | 全局连续 | 跨单元连续 | 每卷独立/连续 | 跨单元连续 |
| 单元分页 | 可选 | 不分页 | 可选 | 强制 | 可选 |
| 在线答题计时 | 逐模块 | 仅整卷 | 逐层+解锁 | 两卷独立 | 自由 |
| 典型场景 | 单元测验 | 标准考试 | 分层测试 | 中考/高考 | 灵活出卷 |

### A — 知识模块
- 单元 = 教学知识块（如"勾股定理"、"实数运算"）
- 纸质格式：单元标题(一级) + 题型标题(二级)，题号跨单元连续
- 在线答题：限时单元逐模块计时提交

### B — 题型（当前"按题型分组"的归宿）
- 单单元（name=""），单元名不显示
- 编辑视图：平表（保留当前"按题型分组"UI）
- 纸质格式：`一、单选题` 为一级标题，连续排版，不分页
- 在线答题：仅整卷计时

### C — 难度递进
- 3个预设单元：基础巩固 / 能力提升 / 拓展挑战
- 纸质格式：单元标题含难度标签（⭐），层级间分隔
- 在线答题：逐层计时 + 逐层解锁

### D — 卷别
- 固定2单元：第I卷(选择题) / 第II卷(非选择题)
- 纸质格式：卷别标题 + 强制分页
- 在线答题：两卷独立计时，不可回退

### E — 通用
- 1个空白单元起步，自由构建
- 与当前"按单元分区"体验一致

## 5. 模板不可变更规则

- **新建时**：Step2 初始展示模板选择器，用户选定后进入编辑
- **编辑时**：模板已锁定，模板选择器显示为只读标签，不可切换
- 不再需要模板切换数据迁移逻辑

## 6. 数据模型变更

### PaperDraft 新增字段

```typescript
interface PaperDraft {
  // ... 现有字段 ...
  template_type: 'knowledge_block' | 'question_type' | 'difficulty_progression' | 'volume' | 'generic';
  // show_units 废弃，由 template_type 决定是否显示单元名
}
```

### 后端 ExamPaper 模型

```python
# exam_papers 表新增列
template_type = Column(String(30), default='generic')
```

### 数据库迁移

```sql
ALTER TABLE exam_papers ADD COLUMN template_type VARCHAR(30) DEFAULT 'generic';
-- 存量数据：show_units=false → 'question_type', show_units=true → 'generic'
UPDATE exam_papers SET template_type = 'question_type' WHERE show_units = false;
```
