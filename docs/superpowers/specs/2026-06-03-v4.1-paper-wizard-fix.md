# V4.1 组卷流程修正

**日期**: 2026-06-03 | **状态**: 已通过

---

## 一、Step2→Step3 结构变更联动

当老师从 Step3 退回 Step2 修改题型数量/分值后，再次进入 Step3：

### 缺口自动补充

`question_config.count > units[].questions.length` → 自动调用 auto-generate 补齐差额。已有题目不动。

### 多余标记提醒

`question_config.count < units[].questions.length` → 题型分区底部显示：

```
已选 5/3 题，超出 2 题（请手动删除多余试题）
```

超出部分试题：红色左边框 + 题号旁「多余」Tag。仅支持删除操作，不支持换题。

---

## 二、"一键选题" 按钮

替换当前「智能生成」按钮。点击行为：

1. 完全清空所有题目
2. 重新调用 auto-generate
3. 展示全新一批题目
4. 仪表盘重置

按钮文案：「一键选题」，图标 ↻

---

## 三、单题换题 [↻] 约束链

换题候选按以下顺序逐级放宽：

```
1. 学科 + 年级 + 知识点 + 题型 + 难度  (严格匹配)
2. 学科 + 年级 + 题型 + 难度              (去掉知识点)
3. 学科 + 题型 + 难度                     (去掉年级)
```

每级检查候选数量，≥1 即停止放宽。候选排除已在试卷中的题目（不重复）。

### 换题成功后追加按钮

换题成功后，在题目行末尾显示「手工选题」按钮：
- 点击弹出抽屉/弹窗
- 可手动搜索/筛选题库
- 选择后替换当前题目

---

## 四、预览格式统一

### 选择题答案排版

每项单独一行：

```
1. 下列属于无理数的是（）                    [简单] 4分 [↻]
   A. -3
   B. 0
   C. √2
   D. 2/3
```

### 输出一致性

预览 / Word导出 / PDF导出 / 打印 — 全部统一此格式。

---

## 五、涉及文件

### 前端
- `steps/RecommendStep.tsx` — 缺口检测+多余标记 / 一键选题 / 换题约束 / 手工选题按钮
- `steps/PreviewStep.tsx` — 选择题答案每项单独一行
- `PaperTemplatePreview.tsx` — 同上
- `PrintPreviewPage.tsx` — 同上

### 后端
- `api/v1/endpoints/exam_papers.py` — swap 端点增加学科/年级/知识点约束链
- `services/recommendation_engine.py` — 无改动（已有难度放宽）

---

## 六、验收标准

1. Step2 减少题数 → Step3 显示多余标记，底部计数正确
2. Step2 增加题数 → Step3 自动补齐缺口
3. 「一键选题」清空全部题目，重新生成新一批
4. 换题 [↻] 候选限定学科/年级范围内，不重复
5. 换题成功后显示「手工选题」按钮
6. 预览选择题答案每项单独一行
7. 预览/导出/打印格式一致
