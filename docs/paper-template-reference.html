D — 卷别

数据模型
单元数: 固定 2
单元名显示: 是

单元1: name="第I卷(选择题)" ⏱30min
  ├ 单选 ×10 2分/题 = 20分
  └ 多选 ×5 4分/题 = 20分
单元2: name="第II卷(非选择题)" ⏱60min
  ├ 填空 ×5 4分/题 = 20分
  └ 解答 ×4 10分/题 = 40分
Step2 编辑视图
卡片式，单元数固定不可增减：

📦 第I卷(选择题) ⏱30min | 40分
单选×10 2分 | 多选×5 4分
[+题型] [×不可删除]
📦 第II卷(非选择题) ⏱60min | 60分
填空×5 4分 | 解答×4 10分
[+题型] [×不可删除]
单元数固定，不可增减
纸质格式
卷别标题 + 强制分页 + 独立题号：

══════ 第I卷 选择题 (40分) ══════
  一、单选题 每题2分 共10题 20分
    1. ...
    ...
  二、多选题 每题4分 共5题 20分
    11. ...
── 新页 ──
══════ 第II卷 非选择题 (60分) ══════
  三、填空题 每题4分 共5题 20分
    1. ... (独立题号)
  四、解答题 每题10分 共4题 40分
    6. ...
在线答题
两卷独立计时，不可回退

第I卷 ⏱30min → 到时自动提交
→ 进入第II卷 ⏱60min
→ 提交第II卷 = 交卷

I卷提交后不可回退
